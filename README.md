# Luanmz07Weekli

> 故事又从第一周开始......

这是一个周历，可以用于记录开学第几周或其他活动

## 安装

### Release

本仓库的`Release`中已经存放好打包的apk，可供直接下载

### 自编译

本程序使用html5开发，使用hbuilderX云打包

可以本地安装Android SDK打包或者使用hbuilder打包

## 使用

下面是程序主页面

![83d32e1a84b89f53c9567108e457851b](https://github.com/user-attachments/assets/421b1149-d0f0-4adf-a5b0-45675e003592)

起始日期为第一周所在日期

选择起始日期后，日期会被自动存储于你的手机
